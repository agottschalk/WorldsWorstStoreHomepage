Welcome to our store

Making a good online store is difficult, especially with so many good competitors out there. So instead, I opted to make a bad one. It seemed like an easier option - if I succeeded, GREAT! I accomplished what I set out to do. If I failed, fine, since I can only do that by creating an actually good store, which is still an achievement in itself. So it was win-win either way.

Actually, this isn't a real store, it's just a mockup of one created for a school project. It looks slick, doesn't it? Okay, so the selection is not very impressive, but the page does have a bunch of cool effects on it. All animations featured were created using the Scriptaculous JavaScript library. Have fun clicking on things and see what happens.
